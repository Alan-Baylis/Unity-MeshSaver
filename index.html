Unity-Meshsaver

A C# Editor Script that allows you to save a usable Mesh .asset file into your project.

View on GitHub Download .zip Download .tar.gz

This is Unity-MeshSaver

Just plop this script into an Editor folder in your project and you can then right-click on MeshFilter components and save meshes from there.
